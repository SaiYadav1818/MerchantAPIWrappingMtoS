Welcome to Payment Gateway Integration

This application provides a complete Spring Boot 3.2 implementation for handling payment gateway redirects with Easebuzz. After payment processing, the gateway redirects users to success or failure pages where all response parameters are displayed dynamically using Thymeleaf templates.
